网站 网页 HTML

网站

网站(website)是指在英特网上根据一定的规则，使用HTML(标准通用标记语言的一个应用)等工具制作的用于展示特定内容相关网页的集合。简单的说，网站是一种沟通工具，人们可以通过网站来发布自己想公开的信息或者利用网站来提供相关的网络服务。人们可以通过网页浏览器来访问网站，获取自己需要的资讯或者享受网络服务。

网页

网页是网站中的一页，通常是HTML格式的文件，它通过浏览器来阅读。

网页是构成网站的基本元素，它通常由图片，链接，文字，声音，视频等元素构成。通常我们看到的网页，常见以.htm或.html后缀结尾的文件，因此将其俗称为HTML文件。

HTML

HTML指的是超文本标记语言，他是用来描述网页的一种语言。

所谓的超文本是指超越了文本的限制(可以加入图片，声音，动画等)，超级链接文本(可以从一个文件跳转到另一个文件，与世界各地主机的文件连接在一起)。

网页形成

前端人员开发代码—–浏览器显示代码(解析，渲染)—-生成最后的Web页面。

常用浏览器

浏览器是网页显示，运行的平台。常用的浏览器有IE(现在的edge)，火狐，Google，Safari，Opera

浏览器内核

浏览器内核（渲染引擎）：负责读取网页内容，整理讯息，计算网页的显示方式并显示页面。

IE—内核Trident—代表IE，猎豹安全，360极速浏览器，百度浏览器

firefox—内核Gecko—代表火狐

safari—内核—代表苹果浏览器

chrome/Opera—内核 Blink（WebKit的分支）—chrome/Opera/新版的edge

国内一般浏览器都是采用webkit/Blink内核，如360，uc，QQ，搜狗等。

图像标签和路径

图像标签（单标签）

在HTML标签中，img标签用于定义HTML页面中的图像。

src是标签的必需属性（此标签的特性），它用于指定图像文件的路径和文件名。

其他属性：

  • 属性–属性值–说明

  • src—图片路径–必须属性

  • alt—文本—替换文本，图像不能显示时的文字

  • title—文本–提示文本，鼠标放到图像上，显示的文字

  • width—像素–设置图像的宽度

  • height—像素–设置图像的高度

  • border—像素–设置图像的边框粗细

    目录文件夹和根目录

    目录文件夹：就是普通文件夹，里面只不过存放了我们页面所需的相关素材，比如imag，HTML文件

    根目录：打开文件夹的第一层就是根目录

过于多的图片时建立的文件夹，查找图片就需采用路径的方式来指定图像文件的位置

相对路径：以引用文件所在的位置为基础 ，而建立出的目录路径，即图片相对于HTML页面的位置

  • 同一级路径 src=“图片名”
  • 下一级路径src=“同一级文件名/图片名”
  • 上一级路径src=“../(返回上一级文件夹)文件名/图片名”

绝对路径：是指目录下的绝对位置，直接到达目标位置，通常是从盘符开始的路径。如src=“c:...”或是网站地址http://…

表格，列表，表单

表格

表格主要用来显示，展示数据，它可以让数据显示的非常规整，可读性好。特别是后台展示数据的时候，能熟练运用表格很重要，一个好的表格能把繁杂的数据表现得有条理。

表格不是用来布局页面，而是用来展示数据。

  • table是用来定义表格的标签
  • tr是定义表格的行，嵌套在table中
  • td是定义表格的单元格，嵌套在tr中
  • 单元格可以放任何元素，文字链接 图片等都可以

表头单元格标签

位于表格第一行或第一列，表头单元格里面的文本内容加粗居中显示。使用th替换td即可。

表格属性

表格标签这部分属性我们实际开发不常用，后面通过css来设置‘

1、HTML table属性

用来定义HTML的表格，具有本地属性 border 表示边框，border属性的值必须为1或空字符串(“”)。该属性不会控制边框的样式，而是由CSS来控制

table元素可以有tr，th，td，thead，tbody，tfoot，colgroup元素

table标签的基本属性如下：

border:可以的取值为1和0,1代表有边框，0代表没有边框。

bordercolor：可以设置边框的颜色，值为颜色值。

bgcolor:设置表格的背景颜色

background:设置背景图片

align：规定表格周围元素的对齐方式，可取（left，center，right）

width：规定表格的宽度（填入百分比或像素值）

2、html tr属性

用来定义表格的一行。由于HTML表格是面向行的，所以必须分别表示每一行

tr元素可以在table，thead，tbody和tfoot元素内使用

tr元素内可以包含一个或者多个td或th元素

它的align,bgcolor等属性已过时，如果要设置属性，请使用CSS设置

3、HTML td属性

用来定义表格单元格，可以同colspan，rowspan，headers局部属性使用

　　(1)colspan： 列跨度，该属性规定了单元格可横跨的列数，该属性的值必须是整数

　　(2)rowspan：行跨度，该属性规定了单元格可横跨的行数，该属性的值必须是整数

　　(3)headers：该属性的值是一个或多个单元的ID属性值，将单元格与列标题相关联，可用于使用屏幕阅读器

注意：每个表格必须包含以上三个元素

4、html th属性

用来定义标题单元格，使我们有效区分数据及其描述

它同 td元素具有相同的局部属性，两者有如下区别：

th是表示头标记，通常位于首行或者首列。而且th中的文字默认会被加粗，而td是不会的

td是数据标记，表示单元格的具体的数据

valign属性可以设置th和td，可以取值Top或者Bottom。

callpadding：内容与单元格边框的间距

cellspacing：防止文本超出边框

如果要使单元格实现跨行或者跨列功能，使用属性colspan和rowspan.

5、HTML thead属性

　　用来定义表格的页眉，表头的包装器。可以定义一行或多行，这些行是 table 元素的列标签

　　没有thead元素，所有的tr被假定为属于表的主

6、HTML tbody属性

　　用来定义表格的主体

7、HTML tfoot属性

　　用来定义标记表格的页脚

注意：

  • thead，tfoot标签内部必须拥有tr标签
  • thead和tfoot标签不管放在table标签内的哪个位置，都会被分别定为到表格的头部和底部。
  • tfoot可以出现在tbody或tr之前或之后。
  • 在html5之前，tfoot元素必须出现在tbody元素之前，在html5中，可以将tfoot元素放在tbody或最后一个tr元素后面

8、html colgroup属性

用来定义表列组，可以使用其来将样式应用于某个列，当然也可以使用下面要说的col元素

具有局部属性 span 的 colgroup表示列组应该横跨的列数。默认是一列，即对表格的一列设置样式colgroup可以包含一个或多个 col元素

9、HTML col属性

用来表示表单个列，建议使用colgroup包裹col元素而不是colgroup直接设置span属性定义组

col也具有局部属性span

col放在的colgroup元素内部，col的咩哥实例表示组中的一列。使用该标签可以将样式应用于列的组和该组的单个列

10、html caption属性

用来定义表格的标题，每个表中只能包含一个元素

表单

表单的目的就是收集用户数据

一个完整的表单包括表单域，表单控件（表单元素），提示信息

表单域

表单域是一个包含表单元素的区域

此标签会将他范围内的表单元素信息交给服务器，以实现用户数据的收集和传递。

表单控件

在表单域中可以定义各种表单元素，这些表单元素就是允许用户在表单中输入或是选择的内容控件。

  • input输入表单元素：用于收集用户数据，单标签

    button 定义可点击按钮（多数情况下，用于通过 JavaScript 启动脚本）。
    checkbox 定义复选框。拥有相同的name属性值
    file 定义输入字段和 “浏览”按钮，供文件上传。
    hidden 定义隐藏的输入字段。

    reset 定义重置按钮。重置按钮会清除表单中的所有数据。
    submit 定义提交按钮。提交按钮会把表单数据发送到服务器。
    text 定义单行的输入字段，用户可在其中输入文本。默认宽度为 20 个字符。

    除了type外还有value（输入字段的初始值），readonly（输入字段只读），disabled（输入字段禁用），size（规定字段尺寸），maxlength（输入字段允许的最大长度），lable（绑定另一个元素 通过id绑定）f
